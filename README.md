# What I learned

HTTP is just text communication that uses newlines (`\r\n` or "registered nurses" lmao) as delimeters.
[RFC 9110](https://datatracker.ietf.org/doc/html/rfc9110) details HTTP semantics.
[RFC 9112](https://datatracker.ietf.org/doc/html/rfc9112) is the HTTP 1.1 specific specification.
